# Commit rules
write commit messages in english.
<br>
every file in different square brackets e.g. [file1.py].
<br>
sign before every file
```
- for deleting files
+ for adding files
# for changing files
```
example commit message:
<br>
```
[+ example.py example file that does examples] 
[- bad_file.py delete bad file]
[# change.py changed thing in change.py file]
```
<br>
also don't touch the backend please
